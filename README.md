# Amazon Sales Analysis – Power BI Dashboard

Project Overview:

This project presents a comprehensive Amazon Sales Performance Analysis built using Power BI.

The objective of this project is to transform raw retail sales data into meaningful business insights through interactive dashboards and data visualization techniques.

The dashboard enables stakeholders to monitor revenue trends, profit performance, product category contribution, regional distribution, and customer behavior.

Business Objectives :

- Analyze overall sales performance
- Track monthly revenue and profit trends
- Identify top-performing product categories
- Evaluate regional sales contribution
- Understand customer rating patterns
- Assess the impact of discount percentage on revenue

Key Dashboard Features :

- KPI Cards (Total Revenue, Total Profit, Orders, Ratings)
- Monthly Revenue & Profit Trend Analysis (Drill-down enabled)
- Category-wise Revenue Distribution
- Region-wise Sales Analysis (Map Visualization)
- Payment Method Contribution
- Discount vs Revenue Scatter Plot Analysis
- Interactive Cross-Filtering Across Visuals
- Drill-Through Navigation for Detailed Insights

Tools & Technologies Used :

- Power BI Desktop
- Data Modeling (Star Schema approach)
- DAX (Data Analysis Expressions)
- Power Query (Data Transformation)
- Data Visualization Best Practices

Dataset Features :

- The dataset includes the following key columns:
- Order ID
- Product Category
- Region
- Discount Percentage
- Total Revenue
- Profit
- Quantity Sold
- Payment Method
- Customer Rating
- Order Date

Key Insights Derived :

- Revenue remains stable with minor monthly fluctuations.
- Profit trends closely follow revenue patterns.
- Revenue contribution is balanced across major product categories.
- Multiple payment methods contribute almost equally to total revenue.
- Discount strategy impact analyzed using scatter plot visualization.
- Regional analysis shows strong international presence.

Skills Demonstrated :

- Data Cleaning & Transformation
- Data Modeling & Relationship Management
- KPI Development using DAX
- Interactive Dashboard Design
- Business Insight Extraction
- Analytical Thinking & Storytelling

Conclusion :

This project demonstrates how Power BI can be effectively used to convert raw business data into interactive visual insights, supporting strategic decision-making and performance optimization.
